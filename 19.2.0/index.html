Introduction

GoCD User Documentation

Welcome to GoCD - an open-source Continuous Integration and Continuous Delivery system.

Quick help

Need a quick start? Setup your first GoCD pipeline, while learning all the concepts, by following the “Getting Started” guide. Make sure you’ve installed the GoCD Server and at least one GoCD Agent.

Read this to setup GoCD pipelines on a Kubernetes cluster.
